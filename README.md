# Sistema de Taller Automotriz

Proyecto estático compatible con Netlify.

## Archivos
- index.html
- styles.css
- app.js
- netlify.toml

## Ejecutar localmente
Con Python:

```bash
python -m http.server 8080
```

Abrir en navegador:

```text
http://localhost:8080
```

## Despliegue en Netlify
Sube el contenido de esta carpeta tal cual. El archivo `netlify.toml` ya deja publicada la raíz.
